Python by Example: Variables

In Python, variables are implicitly declared and used by the interpreter to e.g. check type-correctness. Python will infer the type of variables.

 
a=1
b=2*a
c=3.0
d=7.0/3.0

Variables could be of different types: integers, floats, strings, etc. You can use single or double quotes for strings and you can also define multi line strings.

 
e='This is a single quote string'
f="This is a double quote string"
g='''aaaaa
bbbbb
ccccc'''

Python allows to declare mutiple assignments. 

h = i = a

Print all the variables. 

print(a)
print(b)
print(c)
print(d)
print(e)
print(f)
print(g)
print(h)
print(i)
