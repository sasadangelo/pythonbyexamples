Python by Example: Values

Python has various value types including strings, integers, floats, booleans, etc. Here are a few basic examples with integers and floats.

 
print(1)
print("2*3 =", 2*3)
print(3.0)
print("7.0/3.0 =", 7.0/3.0)

Strings, which can be added together with +. As far as language syntax is concerned, there is no difference in single or double quoted string. Both representations can be used interchangeably. 

print('This is a single quote string')
print("This is a double quote string")
print("String concatenation" + " with double quotes")
print('String concatenation' + ' with single quotes')

Booleans, with boolean operators as you’d expect.

 
print(True)
print("True & False =", True & False)
print("True | False =", True | False)
print("not True =", not True)
