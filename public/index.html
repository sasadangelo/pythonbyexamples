Python by Example

Python is an open source programming language that lets you work quickly and integrate systems more effectively. Python is a clear and powerful object-oriented programming language, comparable to Perl, Ruby, Scheme, or Java. Please read the official documentation to learn a bit about Python code, tools, and modules.

Python by Example is a hands-on introduction to Python using annotated example programs. Check out the first example or browse the full list below.
