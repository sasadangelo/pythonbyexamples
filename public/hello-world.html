Python by Example: Hello World

Our first program will print the classic “Hello World !!!” message. Here’s the full source code.

 
print("Hello World !!!")

To run the program, put the code in hello-world.py and use python3 command.

 
$ python3 hello-world.py
Hello World !!!

Now that we can run a basic Python programs, let’s learn more about the language.
